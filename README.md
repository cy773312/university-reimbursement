# 高校报销文件整理助手

> AI 辅助整理高校学术活动报销材料，支持 Hermes Agent / Claude Code / Codex 等 AI 编程工具。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

---

## 解决的问题

高校报销材料整理是一个典型的**多文件归并 + 逻辑核查**任务：

- 一次讲座/论坛产生大量文件：发票、付款截图、订单、行程单、高速费发票、邀请函、签到表……
- 文件类型杂：PDF 电子发票、手机截图、微信支付凭证、滴滴行程单
- 逻辑关系隐蔽：跨市打车高速费分去程/返程，费用构成需要核对

这个项目把整理规则写成 `AGENTS.md`，让 AI 工具自动执行。

## 快速开始

### 前置条件

- 安装 [Hermes Agent](https://hermes-agent.nousresearch.com)（推荐）或其他 AI 编程工具
- 将本仓库的 `template/` 目录复制到你的报销文件夹

```bash
# 复制模板
cp -r template/* /path/to/your/reimbursement/

# 把原始材料放入 原始文件/ 目录
# AI 会扫描、归类、生成 Excel
```

### 使用

```bash
cd /path/to/your/reimbursement/
hermes -s university-reimbursement
# 或
hermes chat -q "帮我整理报销材料"
```

AI 会自动：
1. 扫描 PDF 和图片文件
2. 识别发票、付款记录、行程单
3. 按一笔费用一个事项归并
4. 生成 `报销整理表.xlsx`

## 工作原理

### 文件归并逻辑

AI 根据以下规则将文件归并为报销事项：

| 依据 | 示例 |
|------|------|
| 金额一致 | 发票 202.01 元 + 付款记录 202.01 元 |
| 日期接近 | 同一日期的打车发票 + 行程单 |
| 收款方相同 | 收款方都是"滴滴出行" |
| 行程关联 | 同一行程的发票 + 行程单 + 高速费 |
| 发票号一致 | 同一电子发票的 PDF + 截图 |

### 跨市打车高速费规则（重要）

跨市行程（如深圳↔珠海）的高速费处理：

1. **高速费归入打车文件夹**，不单独立项
2. 通行费分**去程（载客）** + **返程（空车）** 两部分
3. 司机空车返回的高速费由乘客承担
4. 金额核对：`滴滴发票 + 去程高速费 + 返程高速费 ≈ 实付金额`

### 图片识别

- Hermes 大模型有视觉能力 → 直接用 `vision_analyze` 读图
- 纯文本模型 → 本地 OCR 兜底：`python3 scripts/read_image.py <路径> --brief`

## 覆盖的费用类型

| 日常用语 | 对应场景 | 典型材料 |
|---------|---------|---------|
| 机票 | 飞机出行 | 行程单 + 支付记录 + 订单 |
| 打车 | 网约车 | 发票 + 支付记录 + 行程单 + 高速费 |
| 住宿 | 酒店 | 发票 + 支付记录 + 水单 |
| 高速费 | 通行费 | 归入打车文件夹 |
| 文印 | 打印/印刷 | 发票 + 支付记录 |
| 书籍 | 图书资料 | 发票 + 支付记录 |
| 文具 | 办公用品 | 发票 + 支付记录 |
| 快递 | 邮寄 | 发票 + 支付记录 |
| 茶歇 | 会议茶歇 | 发票 + 支付记录 |
| 劳务费 | 专家讲座 | 发票 + 身份证 + 邀请函 |
| 场地 | 场地租赁 | 发票 + 支付记录 |
| 用餐 | 工作餐 | 发票 + 支付记录 |
