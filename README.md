## 这是什么

一个 AI Agent 工作规则集。把你的报销材料目录交给它，它会：

1. 扫描所有 PDF 和图片，识别每份文件是什么（发票、支付记录、行程单……）
2. 提取关键信息：金额、日期、收款方
3. 按一笔费用一个文件夹归并，统一重命名
4. 生成汇总 Excel，金额逐项核对

半小时的整理工作，压缩到一两分钟。

## 使用方法

### 第一步：安装

```bash
git clone https://github.com/cy773312/university-reimbursement.git
cd university-reimbursement
bash install.sh
```

这个命令会把整理规则安装到你的 AI Agent 中（Hermes 会自动加载，其他 Agent 按各自方式引用 skill 目录）。

### 第二步：准备材料

把这次要报销的所有文件（PDF 发票、微信支付截图、行程单、酒店水单等）放到一个目录下，比如 `~/报销/原始文件/`。

### 第三步：放入项目规则

将仓库中的项目规则文件复制到你的报销目录：

```bash
cp template/AGENTS.md ~/报销/
```

这个文件告诉 AI 你的报销目录里有什么、你想怎么整理。

### 第四步：让 AI 干活

用你的 AI Agent 打开报销目录，然后说：

> 帮我把报销整理一下

AI 会读取规则，扫描文件，开始整理。过程中可能会问你几个问题（比如某张发票不确定归属），确认后继续。几分钟后整理完成。

### 第四步：交付

整理完成后，你的目录里会多出：

```
已整理/                           ← 按费用归类的文件夹
  2026-07-01_机票_北京至上海/
  2026-07-01_打车_机场至酒店/
  2026-07-01_住宿_锦江之星/
  ...
报销整理表.xlsx                   ← 汇总表格，可以直接提交
```

## 核心理念

- **AI 做归类，人做决策**——AI 扫描分析、标记疑问，你来确认。不替你做主。
- **命名让人一眼看懂**——文件夹叫"机票\_北京至上海"，不叫"专家差旅费\_CNY"
- **金额必须对上**——发票 + 高速费 ≈ 实付，对不上就标出来
- **覆盖大多数场景**——通用规则处理常见情况，特殊需求按需补充
