讲座、论坛、会议、差旅、文印……学术报销材料，扔给 AI 自动归类、核验、生成 Excel。

## 这项目解决什么问题

学术报销的痛点不在于财务规则，而在于**整理本身**：

- 一次出差产生十几份文件：发票、微信截图、行程单、高速费票、酒店水单
- 文件类型杂：PDF 发票、截图照片混在一起，命名乱七八糟
- 逻辑关系散：哪张图对应哪笔钱，哪张发票对应哪段行程，全靠人脑硬记

**这不是人该花时间做的事。** AI 擅长的恰恰是这类工作：批量扫描、按规则归类、逐项比对、输出结构化结果。

项目把整理规则写成 AI 能执行的文档，跑一次就知道效果。

## 快速开始

```bash
# 1. 安装
bash install.sh

# 2. 在你的报销目录里放好原始材料，复制模板
cp template/AGENTS.md ~/报销/

# 3. 运行
cd ~/报销 && hermes -s university-reimbursement
```

## 支持的材料类型

| 材料 | 自动识别 | 归入 |
|------|---------|------|
| PDF 电子发票 | 金额、税号、收款方 | 对应费用项 |
| 支付截图 | 金额、时间、商户 | 付款记录 |
| 网约车行程单 | 时间、路线、费用 | 打车费 |
| 高速费发票 | 金额 | 对应打车行程 |
| 酒店水单 | 房费、入住日期 | 住宿费 |
| 图书购物截图 | 金额、书名 | 图书费 |
| 打印清单 | 金额、数量 | 文印费 |

## 设计原则

1. **AI 做归类，人做决策**——AI 扫描文件、按规则分组、标出问题，你来确认，不替你做主
2. **命名让人一眼看懂**——"机票"比"专家差旅费"好懂
3. **金额必须能对上**——发票金额 ≈ 实付金额，对不上就标出来
4. **覆盖大多数场景**——通用规则处理常见情况，特殊需求单独补充
