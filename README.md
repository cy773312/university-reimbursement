## 概述

把一文件夹杂乱的发票、截图、行程单丢给 AI，自动归好类、核好金额、生成 Excel。直接拿去财务处交差。

---

## 痛点

学术报销的烦人之处不在财务规则，在整理本身：

- 一次出差十几个文件：PDF 发票、微信截图、行程单、酒店水单、高速费票
- 文件名全是 `IMG_001.png`、`微信图片_20260526.jpg`、`发票(2).pdf`
- 哪张图对应哪笔钱，全靠人脑硬记

这种事，交给 AI 做。

## 快速开始

```bash
# 1. 安装
bash install.sh

# 2. 把模板放到报销目录，原始材料丢进 原始文件/
cp template/AGENTS.md ~/报销/

# 3. 运行
cd ~/报销 && hermes -s university-reimbursement
```

## 支持的材料类型

| 材料 | AI 做的事 | 归入 |
|------|----------|------|
| 电子发票 | 自动提取金额、税号、抬头 | 对应费用项 |
| 支付截图 | 提取金额、时间、商户 | 付款记录 |
| 网约车行程单 | 匹配发票 + 高速费 | 打车费 |
| 高速费发票 | 归到对应的打车行程 | 打车文件夹 |
| 酒店水单 | 提取房费、入住日期 | 住宿费 |
| 图书订单 | 提取金额、书名 | 图书费 |
| 打印清单 | 提取金额、数量 | 文印费 |

## 核心理念

1. **AI 做归类，人做决策**——AI 扫描、分析、标出问题；你来确认
2. **命名让人一眼看懂**——"机票"比"专家差旅费"好懂
3. **金额必须对上**——对不上就标出来
4. **通用覆盖大多数场景**——特殊需求按需补充
