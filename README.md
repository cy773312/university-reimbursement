## 💡 痛点

学术报销的烦人之处不在财务规则，在整理本身。一次出差十几个文件：PDF 发票、微信截图、行程单、酒店水单。文件名全是 `IMG_001.png`、`微信图片_20260526.jpg`，哪张图对应哪笔钱全靠人脑硬记。

这种事，交给 AI 做。

---

## ✅ 效果

整理前：一个目录、几十个文件、完全乱序。

整理后：

```
已整理/
├── 2026-07-01_机票_北京至上海_1260 元/

│   ├── 02_付款记录_微信支付.png

├── 2026-07-01_打车_机场至酒店_168 元/
│   ├── 01_发票.pdf
│   ├── 02_付款记录.png
```

## 🚀 快速开始

```bash
bash install.sh                       # 安装
cp template/AGENTS.md ~/报销/         # 放模板
cd ~/报销 && hermes -s university-reimbursement   # 运行
```

---

## 📋 支持的材料

| 材料 | AI 做的事 | 归入 |
|------|----------|------|
| 电子发票 | 提取金额、税号、抬头 | 对应费用项 |
| 支付截图 | 提取金额、时间、商户 | 付款记录 |
| 网约车行程单 | 匹配发票和高速费 | 打车费 |
| 高速费发票 | 归入对应打车行程 | 打车文件夹 |
| 酒店水单 | 提取房费、入住日期 | 住宿费 |
| 图书订单 | 提取金额、书名 | 图书费 |
| 打印清单 | 提取金额、数量 | 文印费 |

## 🎯 设计理念

- AI 做归类，人做决策——AI 扫描分析，你来确认
- 命名让人一眼看懂——"机票"比"专家差旅费"好懂
- 金额必须对上——对不上就标出来
- 通用覆盖大多数场景——特殊需求按需补充
