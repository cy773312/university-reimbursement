## 快速开始

将 `template/AGENTS.md` 放入你的报销目录，里面写好了完整的整理规则。然后用 AI Agent 打开这个目录，告诉它"帮我整理报销材料"即可。

支持 Hermes、Claude Code、Codex 等任何能读取和执行 markdown 规则的 AI Agent。

---

## 支持的费用

机票、打车（含高速费）、住宿、文印、图书、快递、茶歇、劳务费、场地、用餐……AI 自动识别文件类型，提取金额、日期、收款方，按一笔费用一个文件夹归并。
